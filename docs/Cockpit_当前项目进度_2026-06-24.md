# MechPilot Cockpit 当前项目进度

## 一、当前项目目标

基于 SolidWorks 2022 COM Add-in，构建 MechPilot 智能辅助插件，提供三大核心能力：
1. **前端 Cockpit** —— AI 助手、图纸审核、快速选型、物料检索、设计计算
2. **本地属性工具** —— 属性填写、属性读取、属性检查、BOM 导出、批量转换
3. **在线选型** —— 通过右侧任务窗格加载在线选型页面

## 三、SolidWorks 插件加载链路

1. **注册表注册**：通过 `scripts/install.bat` 使用 RegAsm 注册 COM Add-in
2. **SolidWorks 启动**：读取注册表 `HKEY_CLASSES_ROOT\SwAgentAddin.Connect\CLSID`
3. **ConnectToSW 调用**：SolidWorks 调用 `ISwAddin.ConnectToSW()`
4. **初始化流程**：
   - SetAddinCallbackInfo2 注册回调
   - GetCommandManager 获取命令管理器
   - LoadConfig 加载配置
   - LoadRules 加载规则
   - InitActionRouter 初始化命令路由
   - AddCommandMgr 注册工具栏按钮
   - AddTaskPane 创建任务窗格

## 六、在线选型入口实现方式

### 调用链路

1. **按钮点击**：用户点击顶部工具栏"在线选型"按钮
2. **SwCmd_OnlineSelection()**：CommandManager 回调方法
3. **ShowOnlineSelectionTaskPane()**：
   - 获取配置的 URL（`GetOnlineSelectionUrl()`）
   - 若任务窗格未创建，调用 `AddTaskPane()`
   - 显示任务窗格 `_taskpaneView.ShowView()`
   - 初始化 WebView2：`_onlineSelectionPane.Initialize(url)`

## 七、配置文件说明

### config.json 关键配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `online_selection_url` | string | `https://xtalpi.aiforce.cloud/app/app_4k6beaevn9gub` | 在线选型页面地址 |
| `cockpit_enabled` | bool | true | 是否启用 Cockpit |
| `cockpit_url_mode` | string | "local" | Cockpit URL 模式 |
| `cockpit_prefer_webview2` | bool | true | 优先使用 WebView2 |
| `auto_show_taskpane` | bool | true | 遗留配置项，当前未接入实际逻辑（ConnectToSW 总是创建任务窗格） |
| `hindsight.enabled` | bool | true | 是否启用 Hindsight 向量检索 |
| `hindsight.base_url` | string | `http://192.168.31.115:8888` | Hindsight 服务地址 |

### 配置加载流程

1. `LoadConfig()` 在 `ConnectToSW()` 中调用（仅插件加载时读取一次）
2. 读取 `config/config.json`
3. 缺失配置项使用硬编码默认值
4. **注意**：修改 `online_selection_url` 等配置后，需重新加载插件或重启 SolidWorks 才生效（点击按钮时读的是内存中的 `_config`，不会重新读文件）

### 错误处理

- **WebView2 初始化失败**：显示中文提示，提供"浏览器打开"按钮
- **页面加载失败**：状态栏显示错误信息，提供刷新按钮
- **导航失败**：`OnNavigationCompleted` 回调处理，显示失败状态
