# MechPilot Cockpit 架构说明

## 一、概述

MechPilot Cockpit 是一个基于 SolidWorks 2022 COM Add-in 架构的智能辅助插件，采用 C# .NET Framework 4.8 开发，提供前端 Cockpit、本地属性工具、在线选型三大核心能力。

## 二、SwAgentAddin 作为 COM Add-in 的职责

### 2.1 核心职责

| 职责 | 说明 |
|------|------|
| **COM 接口实现** | 实现 `ISwAddin` 接口，处理 SolidWorks 加载/卸载生命周期 |
| **CommandManager 注册** | 注册顶部工具栏按钮和回调 |
| **任务窗格管理** | 创建和管理右侧任务窗格 |
| **配置管理** | 加载和管理 `config.json` |
| **命令路由** | 通过 `ActionRouter` 处理前端命令 |
| **上下文收集** | 收集 SolidWorks 文档上下文 |

### 2.2 COM 注册机制

```csharp
[ComVisible(true)]
[Guid("E8F5C9A2-3D14-4E7F-9A1B-C6D5E4F3A2B1")]
[ProgId("SwAgentAddin.Connect")]
[ClassInterface(ClassInterfaceType.AutoDual)]
public class SwAgentAddin : ISwAddin
```

- **ProgId**：`SwAgentAddin.Connect` — 注册表中标识
- **Guid**：固定 UUID — COM 组件唯一标识
- **注册方式**：通过 `scripts/install.bat` 调用 `RegAsm.exe`

### 3.2 按钮注册代码示例

```csharp
// 创建命令组
_cmdMgr.CreateCommandGroup2(
    CommandGroupId,
    "MechPilot",
    "MechPilot tools",
    "MechPilot task tools",
    -1,
    true,
    ref cmdGroupErr);

// 添加按钮（回调方法名必须为 public void）
cmdGroup.AddCommandItem2(
    "在线选型",                              // 按钮显示名称
    -1,                                     // 图标索引
    "打开 MechPilot 在线选型任务窗格",       // Tooltip
    "在线选型",                              // 状态栏提示
    CmdOnlineSelection,                      // Command ID
    "SwCmd_OnlineSelection",                 // 回调方法名
    "",                                      // 状态栏提示（详细）
    CmdOnlineSelection,                      // ImageList 索引
    addItem);                                // 按钮类型（菜单+工具栏）
```

### 3.3 回调机制

- SolidWorks 通过 **反射** 调用回调方法
- 方法签名必须为 `public void MethodName()`
- 方法名必须与注册时指定的名称完全匹配

## 五、WebView2 在任务窗格中的初始化和导航流程

### 5.1 WebView2 初始化流程

```csharp
public async void NavigateOnlineSelection()
{
    if (_webView == null)
    {
        _webView = new WebView2
        {
            Dock = DockStyle.Fill,
            CreationProperties = new CoreWebView2CreationProperties
            {
                UserDataFolder = Path.Combine(
                    SwAgentAddin.GetAddinDirectory(),
                    "online-selection-cache")
            }
        };
        Controls.Add(_webView);
        await _webView.EnsureCoreWebView2Async();  // 异步初始化
        _webView.CoreWebView2.NavigationCompleted += OnNavigationCompleted;
        _ready = true;
    }
    
    if (_ready)
        _webView.CoreWebView2.Navigate(_url);
}
```

### 5.2 关键设计

| 设计点 | 说明 |
|--------|------|
| **异步初始化** | `EnsureCoreWebView2Async()` 避免阻塞 UI |
| **缓存目录** | 使用 `online-selection-cache` 缓存 WebView2 数据 |
| **导航完成回调** | `OnNavigationCompleted` 更新状态栏 |
| **错误处理** | 捕获异常，提供浏览器兜底 |

### 5.3 降级策略

```
WebView2 初始化失败
        │
        ▼
显示中文错误提示
        │
        ▼
提供"浏览器打开"按钮
        │
        ▼
Process.Start(url)
```

    │       │

### 6.2 默认值兜底机制

```csharp
// 示例：online_selection_url 默认值
internal const string DefaultOnlineSelectionUrl = 
    "https://xtalpi.aiforce.cloud/app/app_4k6beaevn9gub";

// URL 规范化时的兜底
internal static string NormalizeOnlineSelectionUrl(string url)
{
    if (string.IsNullOrWhiteSpace(url))
        return DefaultOnlineSelectionUrl;  // 兜底到默认值
    
    // 协议补全
    if (!url.Contains("://"))
        url = "https://" + url;
    
    return url;
}
```

### 6.3 配置项分类

| 分类 | 配置项 |
|------|--------|
| **基础配置** | `execution_mode`, `server_url`, `log_level` |
| **Cockpit 配置** | `cockpit_enabled`, `cockpit_url_mode`, `cockpit_prefer_webview2` |
| **在线选型** | `online_selection_url` |
| **Hindsight** | `hindsight.enabled`, `hindsight.base_url`, `hindsight.bank` |
| **Hermes** | `agent_server.*` |

## 八、前端 Cockpit、本地属性工具、在线选型三类能力的边界

### 8.1 能力边界矩阵

|----------|----------|----------|----------|----------|
| **前端 Cockpit** | CockpitForm (独立窗口) | 本地 + 远程 | Hermes/Hindsight | AI 助手、图纸审核、快速选型、物料检索、设计计算 |

### 8.3 数据流向

| 方向 | 数据内容 | 传输方式 |
|------|----------|----------|
| 前端 → C# | 命令 (ai.material.search 等) | WebView2 PostMessage |
| C# → 前端 | 命令执行结果 | WebView2 PostMessage |
| C# → SolidWorks | 属性读写请求 | COM API |
| SolidWorks → C# | 属性数据、文档上下文 | COM API |
| C# → Hindsight | 检索查询 | HTTP POST |
| Hindsight → C# | 检索结果 | HTTP Response |
| C# → Hermes | AI 调用请求 | HTTP POST |
| Hermes → C# | AI 响应 | HTTP Response |
| TaskPane → 在线服务 | 选型数据 | WebView2 HTTP |

## 九、核心组件职责

### 9.1 组件职责表

| 组件 | 职责 | 关键方法/属性 |
|------|------|--------------|
| **SwAgentAddin** | COM Add-in 主入口 | `ConnectToSW()`, `AddCommandMgr()`, `AddTaskPane()`, `LoadConfig()` |
| **OnlineSelectionTaskPane** | 在线选型任务窗格 | `Initialize()`, `NavigateOnlineSelection()` |

| **AddinConfig** | 配置数据结构 | `OnlineSelectionUrl`, `Hindsight` 等属性（由 SwAgentAddin.LoadConfig 填充） |
| **ActionRouter** | 命令路由 | `HandleCommand()` |
| **LocalToolbeltExecutor** | 本地工具执行 | 构造函数注入依赖，具体执行方法待确认 |
| **HindsightRagClient** | 向量检索客户端 | 构造函数 + 查询方法（具体名称待确认） |
