## 本次更新：本地/远程双模式

### 新增能力

1. **本地模式（Local Mode）**
   - `execution_mode = "local"` 时，Property Fill 直接在 SolidWorks 中写入自定义属性
   - 零件：写入当前零件属性
   - 装配体无选择：写入装配体属性
   - 装配体选中组件：写入选中组件的模型属性
   - 多选时弹出确认框
   - 属性规则从 `rules.local.json` 读取，支持占位符替换

2. **远程模式（Remote Mode）**
   - `execution_mode = "remote"` 保留原有 POST 行为
   - 请求体增强：包含 client 信息、target_scope、local_rules_version
   - 连接失败有明确错误提示

3. **配置扩展**
   - `config.json` 新增：`execution_mode`, `confirm_before_write`, `multi_select_behavior`, `local_rules_file`, `remote_task_endpoint` 等
   - 旧配置仍能加载，缺字段自动用默认值

4. **规则引擎**
   - `rules.local.json` 定义每种文档类型的属性模板
   - 删除后自动生成 demo 规则
   - `RuleProvider` 类负责加载和回退

## 关键经验（历史）

1. 必须实现官方 `SolidWorks.Interop.swpublished.ISwAddin`，不能用手写接口
2. COM GUID `E8F5C9A2-3D14-4E7F-9A1B-C6D5E4F3A2B1` 不要改
3. `ClassInterface` 必须是 `AutoDual`
4. 不能引入 `SolidWorksTools.dll`
5. `ConnectToSW` 中 UI 异常不能导致返回 false
6. `ISelectionMgr` / `IModelDoc2` 从 COM 返回时需要显式 cast
7. `CustomPropertyManager` 是 indexer 属性，用 `get_CustomPropertyManager("")` 调用
